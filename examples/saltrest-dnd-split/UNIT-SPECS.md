# Unit Specs — The Bones of Saltrest (split tier)

## UNIT-001 — The Hook
- Objective: open the adventure — the situation, Saltrest, the call to action.
- Owns: briefs/out/hook.md (`## The Hook`)
- Deps: none
- Criteria-intent: ~150–320 words; uses canon names + concrete sensory detail; a GM
  reads it once and knows what's wrong + what the party is asked to do; seeds ≥2
  investigative threads; no new named NPCs; no dungeon interior / endings.

## UNIT-002 — Cast & Factions
- Objective: the named NPCs + the Drowned Choir, each a motive + a vivid beat.
- Owns: briefs/out/npcs.md (`## Cast & Factions`)
- Deps: none
- Criteria-intent: covers Halrik, Vell, Coria, the Choir; each has motivation + one
  concrete detail; ≥1 playable ally and ≥1 antagonist; Coria framed as mercy-not-malice.

## UNIT-003 — Kelpwater Cove + the clock
- Objective: the 3-level dungeon with stat refs and the ticking-clock rules.
- Owns: briefs/out/dungeon.md (`## Kelpwater Cove`)
- Deps: none
- Criteria-intent: three level subsections; ≥3 `AC/HP` refs + ≥3 `DC <n> <Ability>
  (<Skill>)` checks; the clock as explicit round-by-round rules matching the contract
  (submerge @5, dissolve @10, DC 15); more than one way to engage the Vault.

## UNIT-004 — Resolution
- Objective: the three resolution paths with rewards/consequences.
- Owns: briefs/out/resolution.md (`## Resolution`)
- Deps: UNIT-001, UNIT-002, UNIT-003
- Criteria-intent: three path subsections matching the contract; genuinely different
  end states; each names a concrete reward/consequence; the mercy path references the
  clock's real numbers; consistent with Coria's mercy motive + canon.
